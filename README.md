cheerup
========
[![Build Status](https://travis-ci.org/harijoe/cheerup.svg)](https://github.com/harijoe/cheerup)

## Installation

```
git clone git@github.com:harijoe/symfony-vagrant-up.git
cd cheerup/vagrant
vagrant up
cd ..
composer install
```
You're done, enjoy : http://192.168.33.10
